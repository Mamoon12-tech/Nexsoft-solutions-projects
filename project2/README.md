## 📌 About This Project

A modern, fully responsive SaaS-style landing page built from scratch using only HTML, CSS and JavaScript. No frameworks. No libraries. Just clean, handwritten code.

The landing page is designed for a fictional product called **NexFlow** — a workflow management platform. Every section was carefully structured and styled to feel like a real production website.

---

## ✨ Features

- ✅ Modern hero section with animated background text and live badge
- ✅ Services grid (6 cards) with hover underline animations
- ✅ Pricing section with 3 tiers and a featured "Most Popular" card
- ✅ Call-to-action section with gradient background
- ✅ Fixed navbar with blur backdrop effect
- ✅ Scroll reveal animations using Intersection Observer API
- ✅ Fully responsive — looks great on mobile, tablet, and desktop
- ✅ Hover effects on all interactive elements
- ✅ Optimized page structure and clean semantic HTML

## 🎨 Design Choices

- **Color palette:** Near-black background `#05050a` with orange-red accent `#ff4d00` for energy and contrast
- **Typography:** Syne (display headings) + Inter (body text) — a modern editorial combo
- **Layout:** CSS Grid for the services section, Flexbox for nav and hero
- **Animation:** Scroll-triggered fade-in using `IntersectionObserver` — no jQuery, no GSAP
- **Background text:** Large decorative text using CSS `opacity: 0.02` for depth without distraction
